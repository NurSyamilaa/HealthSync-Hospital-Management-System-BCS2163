# HealthSync

## Course Information
- Course Code: BCS2163
- Course Name: Object Oriented Design & Implementation

## Project Description
HealthSync is a healthcare management system developed as a group project for BCS2163. The project demonstrates object-oriented design and implementation concepts such as encapsulation, inheritance, polymorphism, and abstraction.
